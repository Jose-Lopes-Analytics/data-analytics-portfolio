# 📊 Dashboard Financeiro

Este projeto apresenta uma análise completa de dados financeiros,
com foco em indicadores de desempenho, receitas, despesas e lucratividade.

---

## 🎯 Objetivo

Transformar dados financeiros em insights visuais para apoiar a 
tomada de decisão, facilitando a análise de resultados e tendências.

---

## 📌 Visão Geral do Dashboard

![Dashboard Financeiro](../images/dashboard-financeiro.png)

---

## 📈 Principais Indicadores

- Receita total
- Despesas totais
- Lucro líquido
- Margem de lucro
- Evolução mensal de resultados

---

## 🧠 Insights Gerados

- Identificação de períodos de maior e menor receita
- Análise de variação de despesas ao longo do tempo
- Comparação entre receita e custo operacional
- Apoio à tomada de decisão estratégica

---

## 🛠️ Ferramentas Utilizadas

- Power BI
- Excel
- DAX (quando aplicável)
