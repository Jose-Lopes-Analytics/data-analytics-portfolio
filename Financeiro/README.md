# 📊 Dashboard Financeiro

## 🎯 Objetivo:
Este projeto tem como objetivo analisar indicadores financeiros, como receita,
despesas e lucro, por meio de um dashboard interativo.

## 🛠️ Tecnologias:

- 📊 Power BI  
- 🔄 Power Query 
- 📈 Microsoft Excel  
- 🌿 Git  
- 🐙 GitHub  

## 🗄️ Em estudo:

- SQL Server
- DAX 
- Python

##🛠️ Ferramentas de apoio:

- PowerPoint (apresentação)
- MyColorSpace — paletas de cores: https://mycolor.space/
- Flaticon — ícones: https://www.flaticon.com/
- Instant Eyedropper — captura de cores: https://instant-eyedropper.com/
- ImgBB — hospedagem de imagens: https://pt-br.imgbb.com/

## 📈 Visão geral
Dashboard desenvolvido para acompanhamento de performance financeira de forma 
clara e interativa.

## 📌 Principais indicadores:
- Receita total;
- Despesas totais;
- Lucro líquido;
- Evolução mensal.

## 🚀 Status do projeto:
Finalizado.
