# 🏭 Dashboard de Produção

Este projeto tem como objetivo analisar indicadores de produção, avaliando eficiência operacional, volume produzido e desempenho dos processos.

---

## 🎯 Objetivo

Transformar dados de produção em informações estratégicas para melhorar a eficiência operacional e apoiar a tomada de decisão.
